# 快速分享

快速分享是 Friday 最简单的发布方式——只需一键，你的笔记就能在网上访问。

---

## 什么是快速分享？

快速分享让你可以：
- 🚀 **一键发布** 单篇 Markdown 笔记
- 🔗 **获取链接** 分享给任何人
- ⚡ **即时可用** 几秒钟内完成

---

## 使用前提

- ✅ Friday License 已激活
- ✅ 当前正在查看一个 Markdown 文件

---

## 使用方法

### 步骤 1：打开笔记

在 Obsidian 中打开你想分享的 Markdown 笔记。

### 步骤 2：点击地球图标

在笔记的右上角，你会看到一个 **🌐 地球图标**。点击它。

> [!info] 找不到地球图标？
> 
> 确保你正在查看 Markdown 文件（.md），并且 Friday 插件已启用。

### 步骤 3：等待构建

点击后，Friday 会自动：

1. 打开右侧面板
2. 设置分享模式的 SitePath
3. 将你的笔记构建为网页
4. 准备发布到 MDFriday Share

你会看到进度指示。

### 步骤 4：发布和分享

构建完成后：

1. 浏览器自动打开预览页面
2. 发布选项自动选择 "MDFriday Share"
3. 点击 **发布** 按钮
4. 复制生成的链接并分享

---

## 链接格式

快速分享的链接格式为：

```
https://mdfriday.com/s/{用户目录}/{预览ID}/
```

示例：
```
https://mdfriday.com/s/abc123/xyz789/
```

---

## 注意事项

### 内容可见性

- 快速分享的内容是 **公开可访问** 的
- 任何有链接的人都可以查看
- 不要分享敏感或私密信息

### 链接有效期

- 链接在发布后立即生效
- 目前没有自动过期
- 删除功能正在开发中

### 图片和附件

- 笔记中的图片会自动处理并上传
- 支持 Obsidian 的内部链接格式
- 大文件可能需要更长的构建时间
